# 画像フォルダ

このフォルダには、SUNABACO飯店のウェブサイトで使用する以下の画像を配置してください：

## 必要な画像一覧

1. **hero.jpg** - メインビジュアル画像（推奨: 1920x1080px以上）
2. **about.jpg** - 当館についてのセクション用画像（推奨: 400x300px）
3. **room1.jpg** - シングルルームの画像（推奨: 400x300px）
4. **room2.jpg** - ツインルームの画像（推奨: 400x300px）
5. **room3.jpg** - スイートルームの画像（推奨: 400x300px）

## 画像フォーマット

- JPEG形式（.jpg）推奨
- PNG形式（.png）も使用可

## 配置方法

各画像ファイルをこのフォルダに配置してください。
ファイルパスの形式: `images/ファイル名.jpg`
